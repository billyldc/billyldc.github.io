# 自动化 {#automation}

## 常用软件
笔记管理软件
- [Obsidian](https://obsidian.md/)：用于管理和编写科研相关的笔记, 支持 Markdown 格式.
    - 一定要试试[Latex Suite](https://github.com/artisticat1/obsidian-latex-suite)插件, 可以按自己的习惯写自动扩展文本的代码, 以在 Obsidian 中高效编写 LaTeX 公式和文档.
    - 还可以试试[Folder Notes](https://lostpaul.github.io/obsidian-folder-notes/)插件, 让文件夹也能有自己的笔记页面.
    - 如果用Zotero管理文献, 可以试试[Zotero Integration](https://github.com/mgmeyers/obsidian-zotero-integration)插件, 可以在 Obsidian 中直接插入Zotero文献引用.
- [Trilium](https://github.com/TriliumNext/Trilium)：用于构建个人知识库, 支持层级结构/标签/文件副本等功能.

文献管理软件
- [Zotero](https://www.zotero.org/)：用于管理和组织学术文献, 支持多人合作/多种引用格式和插件扩展.

论文/代码写作
- [git + vscode + LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop)：用于本地编写和编译 LaTeX 论文, 提供语法高亮/自动补全/预览等功能.
    - [copilot for vscode](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot)：用于在 vscode 中使用 AI 辅助编写代码.
    - [codex](https://openai.com/blog/openai-codex)：用于生成和完善代码片段, 提高编程效率.

自动化
- [Mathpix](https://mathpix.com/)：用于将手写或图片中的数学公式转换为 LaTeX 代码, 提高公式输入效率.
- [Alfred](https://www.alfredapp.com/) (macOS)：用于创建自定义快捷键/工作流/代码片段, 提高日常任务的效率.
    - spotlight的加强版, 在一个窗口内集成快速启动应用程序, 搜索文件, 打开网页, 计算器, 自动化脚本等功能.
- [Raycast](https://www.raycast.com/) (macOS)：Alfred的下位替代, 但是剪贴板管理和文件搜索比较好用.

## 一些有趣的例子
- 使用Alfred创建一个快捷键, 可以把DBLP中的文献页面的BibTeX引用一键插入到Zotero中.
- 使用Alfred创建一个快捷键, 把Obsidian中的所选文字发给AI进行总结, 并把结果插入剪贴板.
- 使用mathpix把论文的pdf转换为markdown格式, 并用Obsidian管理.
- 使用AI检查论文中的语法错误和数学正确性.

# Automation {#automation}

## Common software
Note-taking software
- [Obsidian](https://obsidian.md/): used to manage and write research-related notes, supports Markdown.
    - Try the [Latex Suite](https://github.com/artisticat1/obsidian-latex-suite) plugin to define custom text expansions for efficiently writing LaTeX formulas and documents in Obsidian.
    - Also try [Folder Notes](https://lostpaul.github.io/obsidian-folder-notes/), which gives folders their own note pages.
    - If you use Zotero for references, try [Zotero Integration](https://github.com/mgmeyers/obsidian-zotero-integration) to insert Zotero citations directly into Obsidian.
- [Trilium](https://github.com/TriliumNext/Trilium): for building a personal knowledge base; supports hierarchical structure, tags, and file copies.

Reference management software
- [Zotero](https://www.zotero.org/): for managing and organizing academic literature; supports collaboration, multiple citation styles, and plugin extensions.

Paper/code writing
- [git + VS Code + LaTeX Workshop](https://marketplace.visualstudio.com/items?itemName=James-Yu.latex-workshop): for local authoring and compiling LaTeX papers; provides syntax highlighting, autocomplete, and preview.
    - [Copilot for VS Code](https://marketplace.visualstudio.com/items?itemName=GitHub.copilot): AI-assisted coding and paper proofreading within VS Code.
    - [Codex](https://openai.com/blog/openai-codex): for generating and refining code snippets to improve programming efficiency.

Automation
- [Mathpix](https://mathpix.com/): converts handwritten or image math into LaTeX code to speed up formula input.
- [Alfred](https://www.alfredapp.com/) (macOS): create custom shortcuts, workflows, and snippets to streamline daily tasks.
    - An enhanced version of Spotlight, integrating quick app launching, file searching, web browsing, calculator, automation scripts, and more in one window.
- [Raycast](https://www.raycast.com/) (macOS): a modern alternative to Alfred with particularly good clipboard management and file search.

## Some interesting examples
- Create an Alfred shortcut to insert BibTeX citations from DBLP directly into Zotero.
- Create an Alfred shortcut to send selected text from Obsidian to AI for summarization and copy the result to the clipboard.
- Use Mathpix to convert a paper's PDF to Markdown format and manage it with Obsidian.
- Use AI to check for grammatical errors and mathematical correctness in papers.
